# Projeto de Microeletrônica
## *Projeto em grupo - Disciplina: Microeletrônica*
- Projeto completo de uma fonte linear estabilizada AC–DC, envolvendo dimensionamento elétrico (tensão, corrente e potência) de todos os componentes, incluindo ponte retificadora de 4 diodos, transistor de passagem e diodo Zener, com análise de custo e escolha baseada em custo-benefício.

- Simulação do circuito no LTSpice para validação dos valores calculados, seguida de montagem em protoboard, testes experimentais com osciloscópio e comparação entre valores teóricos, simulados e reais.

- Atuação como líder na organização e execução do projeto, incluindo soldagem em perfboard, testes finais do circuito e documentação técnica, simulando um fluxo completo de desenvolvimento de produto em ambiente industrial.
